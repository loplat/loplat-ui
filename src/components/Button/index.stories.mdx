# Button

`Button`컴포넌트의 크기는 `children`의 내용과 `size`프로퍼티에 따른 `padding`값에 의해 결정됩니다.

<Canvas>
  <Story name="default" args={{ children: 'default', role: 'button' }}>
    {runTest(Template.bind({}))}
  </Story>
</Canvas>
