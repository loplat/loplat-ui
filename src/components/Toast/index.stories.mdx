## 사용방법

1. `app.js`에 `<Toast />`을 작성해주세요.

```js
import { Toast } from 'loplat-ui';

function App() {
  return (
    <div>
      // ...
      <Toast />
    </div>
  );
}
```

2. `toast`컴포넌트를 사용해야하는 파일에서 아래처럼 작성해주세요.

- `toast.success`, `toast.danger`, `toast.warning`, `toast.info` ,`toast.white`로 토스트색상을 변경할 수 있습니다.

```jsx
import { toast } from 'loplat-ui';

const SomeComponent = () => {
  return (
    // code ...
    <Button color="primary2" onClick={() => toast.success('토스트 내용을 작성해주세요')}>
      성공!
    </Button>
  );
};
```
