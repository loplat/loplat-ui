# RadioButton

`<RadioButton />`는 관련된 옵션을 나타내는 라디오 그룹에서 사용합니다.

임의의 그룹 내에서는 동시에 하나의 라디오 버튼만 선택할 수 있습니다.

`input[type='radio']` 의 속성을 계승합니다.
