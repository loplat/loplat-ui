# Underline

`SerializedStyles`를 적용하여 밑줄을 긋습니다.

<Canvas>
  <Story
    name="underline"
    argTypes={{
      borderRadius: {
        description: '`underline`의 border-radius 입니다.',
        ...generateTable({ defaultValue: '2px', type: 'string' }),
      },
      bottom: {
        description: '`underline`이 요소의 bottom과 떨어져있는 정도입니다.',
        ...generateTable({ defaultValue: '-8px', type: 'string' }),
      },
      color: {
        description: '`underline`의 색상입니다.',
        control: 'color',
        ...generateTable({ defaultValue: '#0c8efc' }),
      },
      height: { description: '`underline`의 높이입니다.', ...generateTable({ defaultValue: '3px', type: 'string' }) },
      width: {
        description: '`underline`의 너비입니다.',
        ...generateTable({ defaultValue: '10px', type: 'string' }),
      },
    }}
  >
    {(args) => <Underline {...args} />}
  </Story>
</Canvas>

<ArgsTable story="underline" />

## 사용 방법

`underline`은 `@emotion`의 `SerializedStyles`를 return 하는 함수입니다.

적용하기 위해서는 `underline`를 사용하는 styled 컴포넌트를 생성해야 합니다.

```js
import { styled, underline } from 'loplat-ui';

const Underline = styled.span`
  ${underline({ borderRadius, bottom, color, height, width })};
`;
```
