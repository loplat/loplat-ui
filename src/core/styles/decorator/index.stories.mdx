## 사용 방법

underline은 `@emotion`의 `SerializedStyles`를 return 합니다.

```js
import { styled, underline } from 'loplat-ui';

const Text = styled.span`
  ${underline({ borderRadius, bottom, color, height, width })};
`;
```
